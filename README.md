# Fraud Detection App

A Machine Learning-based web application that predicts whether a financial transaction is fraudulent using Streamlit and Scikit-learn.

## Features
- Fraud prediction
- User-friendly Streamlit interface
- Trained ML pipeline
- Fast predictions

## Technologies
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Joblib

## Installation

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
streamlit run Fraud_detection_app.py
```
